AEM Rules for SonarQube

What is AEM Rules for SonarQube? It is a set of rules in a form of a SonarQube plugin detecting possible bugs and bad smells specific for AEM development in our code. Plugin presented for the first time on Connect Web Experience conference in Basel 2015.

Features

Check your code against AEM-specific violations
